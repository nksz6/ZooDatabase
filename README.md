# Zoo Database Management System
A Java Swing-based application for managing zoo operations

## Features
- Task Management
- Location Management
- Employee Management
- User-friendly GUI

## Prereqs
- Java 21
- NetBeans IDE
- MySQL (8.0+)

## Database Setup
1. Install MySQL
2. Create a new database
3. create the tables
4. clone the repo
5. update the connection details in DatabaseConnection.java
6. Run in NetBeans
